CSS has following text properties:

  • text-align
  • text-align-last
  • text-decoration
  • text-decoration-color
  • text-decoration-style
  • text-decoration-line
  • text-indent
  • text-justify
  • text-overflow
  • text-shadow
  • text-transform

Examples

text-decoration property specifies the decoration added to the text. It is shorthand for three properties. text-decoration-line is required. It sets different types of text decoration lines. text-decoration-line has these three values: underline, overline, and line-through.

It has one more value that is - none. text-decoration style has these values: solid, double, dotted, dashed, wavy .

text-indent property indents the first line with different values. We can specify length of indent in pixels as well as as percentage. text-justify property justifies the text. text-overflow property specifies how overflowed content that is not displayed should be provided to the user. It can be clipped, display an ellipsis (...), or display as custom string. text-justify property has these values - auto, inter-word, inter-character, and none; text-shadow property adds shadow to the text. text-shadow property requires horizontal shadow, vertical shadow, blur radius, and shadow color. Horizontal shadow and vertical shadow are required but blur radius and shadow color are optional. text-transform property controls the capitalization of text. It has these values - none, capitalize, uppercase, lowercase.
